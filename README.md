
This code repository displays all programs used to complete this Dissertation.

There are four main codes:

1) The Centre-of-Inertia (COI) analysis program. This program uses:
   -An iterative COI-based approach to calculate the total frequency response of all the synchronous generators of the system.
   -A calculation of the total effective system inertia constant.
   
2) The Spatial Analysis Program . This program uses:
   -An iterative approach to measure the frequency response at EACH ZONE using BB1.1.
   
3) The Results analysis Program. This Program uses:
   - Data Processing techniques using Pandas and NumPy to sort data into Dataframes and categorise stable vs unstable responses.
   - Matplotlib to formulate results into tables and graphs.
